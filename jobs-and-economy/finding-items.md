This command allows players to search every shop on the server for a specific item.

#### **Usage**

```
/cshop finditem tobuy <item name>
/cshop finditem tosell <item name>
```

### **What This Command Does**

* Searches all **player shops** and **admin shop entries**.
* Shows all sellers offering that item.
* Displays:
  * Shop owner name
  * Price
  * Quantity in stock
  * Buy/Sell options
  * Warp button to teleport directly to the shop (if using Player Warps)
* Results appear in a GUI for easy browsing.
